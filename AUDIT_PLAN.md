## DrumDrums Offline Shelf Rollout
- Injected the DrumDrums Offline shelf (download / install tips / dismiss UI) plus the `file://` library-link conversion helper into the library homepage and every app so each DrumDrum can be saved as a standalone `.html`.
- The library home now defaults to the minimized download icon (no popup), and all other DrumDrums collapse into the same icon when you dismiss the helper so it can be reopened on demand without covering UI.
- Added safe-area padding for the highest-risk layouts (`index.html`, `timezone-converter`, `pomodoro`, `parking-timer`, `focus-timer`) so the floating shelf never overlaps controls, even on compact mobile screens.
- Automated slugged filenames and inlined Blob downloads so offline copies keep their entire HTML/JS/CSS in one file with no remote dependencies.
- Validation: `node tests/offline-module-smoke.test.js` simulates downloads and link replacement for `index.html`, `timezone-converter`, and `pomodoro`, asserting Blob payloads, URL revocation, and that converted buttons preserve attributes + keyboard access.
- Manual spot check guidance:
  1. Open any DrumDrum in a browser, hit “Save offline (.html)” and confirm the downloaded file name matches the page title slug.
  2. Launch that saved file via `file://` and inspect that library links become inline notice buttons instead of navigating.
  3. Dismiss the shelf and ensure your controls remain unobstructed—on mobile, scroll to see the extra padding doing its job.
